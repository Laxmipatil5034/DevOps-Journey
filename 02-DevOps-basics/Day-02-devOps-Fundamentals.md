# Day 02 - DevOps Basic Fundamentals

## What is DevOps?
DevOps is a culture and set of practices that combines Development and Operations teams to deliver software faster and more reliably.

## Why DevOps?
- Improves collaboration
- Automates repetitive tasks
- Speeds up software delivery
- Reduces errors and downtime

## Benefits of DevOps
- Faster deployments
- Better software quality
- Quick bug fixing
- Continuous feedback

## CI/CD Basics
CI - Continuous Integration
CD - Continuous Delivery / Continuous Deployment

Code → Build → Test → Deploy

## DevOps Lifecycle
Plan → Develop → Build → Test → Release → Deploy → Operate → Monitor

## Key Takeaways:
- DevOps improves collaboration between Dev and Ops teams.
- CI/CD automates software delivery.
- DevOps Lifecycle consists of Plan, Develop, Build, Test, Release, Deploy, Operate, and Monitor.
