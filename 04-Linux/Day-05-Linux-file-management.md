## Commands Practiced

### List Files
bash
ls
ls -l

### Change Directory
cd Documents
cd ..

### Present Working Directory
pwd


### Create Directory
mkdir DevOps
mkdir cloud


### Delete Files & Directories
rm file.txt
rm -r folder
rm -rf folder
rmdir EmptyFolder

### View File Contents
cat myfile.txt
zcat

### Create File
touch demo.txt
### View First Lines
head myfile.txt

### View Last Line
tail myfile.txt
tail 
tail -f myfile.txt


### Read Large Files
less newfile.txt
more newfile.txt


### Copy Files & Directories
cp myfile.txt newfile.txt
cp -r /devops /cloud


### Move & Rename Files
bash
mv myfiles.txt devops/
mv myfile.txt newfile.txt
mv devops/ linuxfor_devops

- Practiced real-world Linux file management.
