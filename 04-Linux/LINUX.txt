Day 4 – Linux Introduction Notes

1. How Does the Internet Work?

The internet is a network of connected computers.
Data is transferred using TCP/IP.
A browser sends a request, and a server sends back the response.

2. What is a Server?
A server is a computer that provides services or data to clients over a network.

3. Web Server vs Application Server?
Web Server
Serves static content (HTML, CSS, Images).
Examples: Apache, Nginx.
Application Server
Runs business logic and generates dynamic content.
Examples: Tomcat, JBoss.

4. Types of Applications?
Standalone Applications
Web Applications
Mobile Applications

5. Standalone Application?
Runs on a single computer.
Does not require the internet.
Example: Calculator, MS Paint.

6. Web Application?
Runs in a web browser.
Requires internet/network.
Example: Gmail, Amazon, YouTube.

7. Application Support?
Monitoring applications.
Resolving issues.
Ensuring smooth operation.

8. Application Maintenance?
Fixing bugs.
Updating software.
Improving performance and security.
